## Description
The CLI-utility where you can play 5 simple math games.

## Installation
```
git clone git@github.com:timosii/brain-games.git
make build
make package-install
```

## Usage
Use one of this commands:
```
brain-even
brain-calc
brain-gcd
brain-progression
brain-prime
```
